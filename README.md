*Nombre del proyecto*

Generador Seguro de Contraseñas

Nombre del desarrollador

Jhosser Yail Guevara Cuenca

Objetivo

Desarrollar una aplicación en Python capaz de generar contraseñas seguras y personalizadas, aplicando los conocimientos adquiridos sobre lógica de programación y contribuyendo a mejorar la seguridad digital de los usuarios.

Funcionalidades
Generación automática de contraseñas.
Selección de longitud.
Inclusión de mayúsculas.
Inclusión de números.
Inclusión de caracteres especiales.
Generación aleatoria utilizando Python.
Fecha

Junio de 2026.

Introducción

Las nuevas tecnologías forman parte de la vida diaria y cada vez más información personal se almacena en plataformas digitales. Esto hace que la seguridad de las cuentas sea fundamental. Este proyecto propone una herramienta sencilla que permite generar contraseñas seguras, aplicando los conocimientos desarrollados durante la asignatura de Lógica de Programación.

Descripción del problema

Muchas personas utilizan contraseñas fáciles de adivinar o repiten la misma contraseña en diferentes servicios, lo que incrementa el riesgo de sufrir ataques informáticos y pérdida de información personal. Por esta razón, es importante contar con herramientas que permitan generar contraseñas más seguras.

Relación con la asignatura

Durante el desarrollo del proyecto se aplicaron los contenidos estudiados en las cuatro unidades del curso, incluyendo la resolución de problemas, el manejo de datos, el diseño de algoritmos mediante diagramas de flujo, las estructuras condicionales, los bucles y la organización del código utilizando GitHub como herramienta de control de versiones.

Explicación del sistema

El programa solicita al usuario la longitud de la contraseña y permite seleccionar si desea incluir letras mayúsculas, números y caracteres especiales. Posteriormente, construye el conjunto de caracteres disponibles y, mediante un ciclo repetitivo, genera una contraseña aleatoria que finalmente es mostrada en pantalla.

Reflexión sobre el impacto de la tecnología

El desarrollo de este proyecto permitió comprender que la programación puede utilizarse para resolver problemas reales de la sociedad, como la protección de la información personal. A medida que la tecnología avanza, también aumentan los riesgos relacionados con la seguridad digital, por lo que herramientas como este generador de contraseñas pueden fomentar mejores prácticas y crear mayor conciencia sobre la importancia de proteger los datos personales.
